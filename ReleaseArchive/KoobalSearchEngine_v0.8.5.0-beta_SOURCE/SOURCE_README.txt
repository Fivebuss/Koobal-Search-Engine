Koobal Search Engine — Source Distribution
============================================

Version:  v0.8.5.0-beta
Git tag:  v0.8.5.0-beta (commit d18205d)
License:  MIT (see LICENSE)

This folder contains the complete C# source used to build the v0.8.5.0-beta
plugin assembly shipped in the companion binary zip:

  KoobalSearchEngine_v0.8.5.0-beta.zip

This package is source only — no compiled DLL is included here. See BUILD.txt
for rebuild instructions and README.txt for install/usage scope.

BrandingAssets/ contains marketing PNGs stored with the source tree. They are
NOT required at runtime (the in-game wordmark is drawn programmatically).

Maintainer notes: RELEASE_PROCESS.md and ROLLBACK.md are optional context for
future releases; forum readers can ignore them.
